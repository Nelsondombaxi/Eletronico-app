# Eletrônico App 🎧

Projeto antigo de uma landing page para um app de música eletrônica, feito como experimento visual com efeito parallax, player fixo e interface inspirada em plataformas de streaming.

> ⚠️ Este projeto foi desenvolvido há um tempo como parte do meu aprendizado em HTML, CSS e estrutura de páginas. Será mantido aqui como referência da minha evolução.

---

## 🔥 Funcionalidades

- Layout responsivo com imagem de fundo parallax
- Ícones do Font Awesome para navegação moderna
- Seções de "Em Destaque" e "Playlists"
- Barra de player fixo com controles visuais
- Estilo escuro com gradientes e sombras
